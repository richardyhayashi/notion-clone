## Project 

### Create Notion Clone

`$ mkdir src`
`$ cd src`
`$ npx create-next-app@latest --ts .`
or
`$ npx create-next-app@latest --ts src`
`$ cd src`

### Run Notion Clone

In `src` foler:
`$ npm run dev`

### Shadcn UI

#### Init Shadcn UI

`$ npx shadcn@latest init`

#### Add Component

`$ npx shadcn@latest add {component}`

### Clerk

`$ npm i @clerk/nextjs`
